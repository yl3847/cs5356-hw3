My Project Experience

Natural Language Processing Project: Sentiment Analysis & Text Classification

Cornell University NLP Project, NYC | Feb 2025 - Mar 2025

  • Developed and implemented a Perceptron model and Multilayer Perceptron (MLP) for text classification tasks, specifically Sentiment Analysis (SST-2) and Newsgroup Classification.
  • Preprocessed data and engineered features using Bag of Words (BoW), n-grams, and custom features (negation, intensifiers).
  • Implemented Perceptron from scratch, utilizing a linear classification approach with feature-based weight updates.
  • Designed and trained an MLP model using PyTorch, including embedding layers, hidden layers with various activation functions (ReLU, Sigmoid, Tanh, Leaky-ReLU), and dropout for regularization.
  • Optimized model performance through hyperparameter tuning, batching, and margin-based updates.
  • Conducted error analysis to identify and address model misclassifications, focusing on issues like negation, sarcasm, and ambiguous context. Benchmarked model performance with different batch sizes.

Sentiment Analysis with Semi-Supervised Learning on Text Data

Cornell University Machine Learning Project, NYC | Oct 2024 - Nov 2024

  • Implemented advanced semi-supervised learning techniques (DBSCAN, Label Spreading, and Self-Training) to augment 60% unlabeled data, increasing model training data volume by 2.5x.
  • Designed and compared supervised classifiers (MultinomialNB, SGDClassifier, Random Forest, XGBoost, LightGBM) to achieve a test accuracy of 95.4% using a Voting Classifier ensemble model.
  • Utilized TF-IDF vectorization with n-grams and optimized data dimensionality (top 5000 features) to capture rich textual context while reducing overfitting risk.
  • Applied clustering methods (DBSCAN, K-Means, Gaussian Mixture Models) with Python libraries like scikit-learn and NLTK, achieving the best performance with DBSCAN for labeling noisy and high-dimensional text data.

Analyzing the Impact of Air Quality on Health Outcomes Using Machine Learning

Zhejiang University Design and Research Institute, Urban Science and Analytic Intern, Hangzhou, China | June 2022 - Aug 2022

  • Integrated multi-source datasets (O3, PM2.5, COPD, Asthma, Household Income) for 10 years, conducted feature engineering and outlier detection using z-scores, ensuring clean, high-quality inputs for downstream analysis. Normalized data with StandardScaler for clustering and regression tasks.
  • Applied K-Means and HDBSCAN clustering algorithms to identify patterns in air pollution and respiratory health outcomes, reducing feature dimensions with PCA for effective visualization.
  • Performed SHAP analysis to interpret model outputs, revealing that PM2.5 was the dominant predictor for asthma rates while household income exhibited a mitigating effect.
